# Preisspion Webseite

Enthalten sind:

- `index.html`
- `style.css`
- `impressum.html`
- `datenschutz.html`
- `assets/preisspion-logo.jpeg`

## Vor dem Veröffentlichen

1. In `index.html` nach `DEIN-WHATSAPP-KANAL-LINK` suchen und durch den echten WhatsApp-Kanal-Link ersetzen.
2. Falls Google Analytics verwendet werden soll, den eigenen GA4-Code im `<head>` von `index.html` ergänzen.
3. Erst nach Einrichtung der Domain eine Datei `CNAME` mit der Domain anlegen.
4. Prüfen, ob die Angaben zu Hosting und Cookies in der Datenschutzerklärung zur tatsächlichen technischen Umsetzung passen.
